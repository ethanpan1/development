Development

For this assignment, I created a list interface using React. I chose to create a cheese shop called La Fromagerie, with the list interface serving as a display case for the cheeses on sale while allowing users to add cheese to a shopping cart. The goal of the application is to help users know about the cheeses they're buying (milk source, firmness) and provide an easy and clear interface for buying their cheese. The ability to filter and sort the cheese, as well as the seamless (indeed, one page) shopping interface, makes this app valuable to users as they peruse this imaginary shop.

Usability Principles

I considered a number of usability principles while designing this interface.

  • Interaction: filters, sorts, and buttons are clearly labelled; correct interaction is obvious (e.g., hover states)
  • Layout: the cart is separated from the display case; nice flow from filtering/sorting, to adding to cart, to checking out.
  • Text: information for each cheese is succinctly presented and displayed in both the display case and the cart.
  • Accessibility: color choices and font/image size are readable; image alts are provided.
